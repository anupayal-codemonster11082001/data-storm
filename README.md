**Data Storm**
A simple server application that uses MongoDB Atlas Clusters to store, show and delete entires
<img width="1919" height="960" alt="image" src="https://github.com/user-attachments/assets/a5fc3654-7176-41d3-98d3-cc6fe0d65782" />

**Routes**
/ => a simple form that executes the POST method
/post => displays all entries
/delete/:id => deletes entry if id parameter
/latest/:N => returns top N latest entries
